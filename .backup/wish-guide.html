Wish Guide — Pull Efficiently

[Ad]

Pity System (Community-Verified)

  • Hard pity: 90 wishes guarantees a 5★
  • Soft pity: Starting at wish 74, 5★ rate increases ~6% per wish (community-tested)
  • 50/50: First 5★ has 50% chance to be featured. If you lose, next is 100% guaranteed.
  • Weapon Banner: Epitomized Path requires up to 240 wishes for guarantee. Never pull as F2P.

FAQ

Should I pull C1 or signature weapon?

Usually C1 unless the weapon banner has both featured weapons that you want. Constellations are guaranteed (180 max). Weapon banner pity can reset if you get the wrong featured weapon. Characters > weapons for F2P.
